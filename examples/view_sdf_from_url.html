View SDF from URL

This example shows how to visualize an SDF file on the browser by passing the URL of its resources.

The SDF parsing happens completely on the browser, so there is no need for a GzWeb server to be runnning in the backend.

You can test it as follows:

  1. Add full URLs to all resources needed by the model (meshes, textures, etc), one by one, if any
  2. Fill the full URL to the model's SDF file
  3. Press "Load model!"
